LinkOS

2022-05-18 OS 评论

概述

Link OS就是一颗万精油，目的是希望 跨平台/跨协议 统一接入平台，开发者只需要做一小部分适配即可接入物联网的一个系统。

1. esp-idf

- esp工具的下载地址很多，常见的有：  
[github](https://github.com/espressif/esp-idf)  
[gitee](https://search.gitee.com/?skin=rec&type=repository&q=esp-idf)

2. ubuntu

- 自带的是2.7和3.6的python，ESP-IDF supports Python 3.7 or newer ，所以需要考虑安装3.7环境
- 增加python更新源  sudo add-apt-repository ppa:deadsnakes/ppa
- 更新 apt update
- 安装 apt install python3.7 -y
- 删除软链接 rm -rf /usr/bin/python3
- 创建软链接 ln -s /usr/bin/python3.7 /usr/bin/python3
- 检查是否安装成功  python3 --version

3.pip

- apt install python3-pip
